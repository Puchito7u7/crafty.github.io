Tải về

Windows

  1. Nhấn vào nút bên dưới để tải

  2. Mở trình cài đặt và làm theo hướng dẫn

macOS

cho nhân Intel (x64) hoặc bản Rosetta

  1. Nhấn vào nút bên dưới để tải file chạy (disk image)

  2. Mở file và cài đặt ứng dụng. Lỗi thường gặp

AUR

cho hệ điều hành thuộc Arch Linux: Manjaro, SteamOS, ...

Tải trình chạy từ AUR theo nhiều cách

Nếu bạn dùng yay làm trình cài AUR, thử dùng lệnh sau:

yay -S tlauncher

Flathub

cho hệ điều hành thuộc Linux dùng Flatpak

Mở Flathub và làm theo hướng dẫn

Để tắt chức năng đăng nhập bằng tài khoản Microsoft, dùng lệnh sau:

flatpak --user override ch.tlaun.TL --env=TL_BOOTSTRAP_OPTIONS="-Dyoutu.be=dQw4w9WgXcQ"

Ubuntu

và các hệ điều hành thuộc Debian: Ubuntu, ...

  1. Nhấn nút bên dưới để tải file đóng gói (.deb)

  2. Mở file đóng gói bằng "Software Install"

  3. Cài file đóng gói

  4. Khi cài đặt hoàn thành, có thể mở trình chạy

Cài thủ công bằng gói Java

  1. Tải và cài đặt phiên bản mới nhất của Java LTS theo hệ điều hành của bạn (nên dùng Adoptium).
  2. Tải file đóng gói trình chạy và khởi chạy. Nó sẽ tự cài mọi thứ cần thiết.

Đây là bản lậu à?

Minecraft vốn là game trả phí.

Việc bản lậu Minecraft tồn tại là không thể không có.

Bạn có thể chơi đơn, hoặc chơi online (chỉ riêng với các máy chủ game lậu). Một số tính năng chỉ người chơi trả phí mới dùng được.

Nếu bạn thấy game hay, hãy mua nó ;)
